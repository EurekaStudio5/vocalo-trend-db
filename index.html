      このサイトについて

      ボカロトレンドDBとは

      YouTube上のボカロ（VOCALOID / SynthV / CeVIO など）オリジナル曲のトレンドを毎日自動で集めて見える化する、非公式のファンサイトです。

      こんな人向け

      • ボカロが好き — いま伸びてる新曲をまとめてチェック。サイト上でそのまま聴けます。
      • ボカロPになりたい・バズりたい — どのボーカル・どんな長さ・どんなタイトルが伸びやすいか、データで分かります。

      データについて

      • YouTube上の公開情報を毎日1回自動収集しています。
      • ランキングは「期間内（週間=7日 / 月間=30日 / 年間=1年 / 全期間）に投稿されたボカロ関連オリジナル曲」を再生数順に並べたもの。歌枠・カラオケ配信・歌ってみた・メドレーは除外しています。
      • ボカロP名鑑は「登録者1万人以上」または「500万再生超のボカロ曲を持つ」チャンネルを掲載。YouTube上のランキング・チャンネル検索から日々自動追加されます。
      • 再生数・登録者数は一部概数（万単位など）の場合があります。ランキング上位曲は正確な値を取っています。

      VocaScore（曲単体評価スコア）とは

      チャンネル登録者数や投稿日時に左右されない「曲そのものの評価」を0〜100で出す独自スコアです。ポイントは、固定の基準ではなく「同じ条件の曲の期待値と比べてどうか」で採点していること。すべてのランクイン曲からデータの傾向を毎日学習し、相対評価（パーセンタイル）で点数化します。

      • 好感度（40点） — 高評価率（高評価数÷再生数）の高さ。ただし高評価率は再生数が伸びるほど自然に下がる傾向がある（ライト層に届くため）ので、同じ再生数帯の曲の期待値と比べて高いかどうかで採点します。再生数が多い曲が損をしない設計。
      • 拡散力（30点） — 再生数の大きさ。ただし登録者が多ければ再生数が多いのは当たり前なので、同じ登録者規模のチャンネルの期待値と比べてどれだけ上振れているかで採点します。大手も新人も「自分の規模での期待値」と比べるのでフェア。
      • 勢い（30点） — 投稿からの経過時間でならした再生数（再生数÷日数^0.65）。昨日出た曲と半年前の曲をフェアに比べるための補正です。

      グレードは S（80+）/ A（65+）/ B（50+）/ C（35+）/ D。Sは「3要素すべてで上位」の曲だけが取れるレアもの。取れなかった指標がある曲は、取れた分だけで100点満点に換算します。Scoreバッジをタップ（PCはカーソルを乗せる）と内訳が見られます。

      Pランキングの軸について

      「登録者数」のような王道から「曲名のひらがな率」のような変化球まで約80種類。各Pの直近の投稿動画・ランクイン曲・チャンネル情報から自動計算しています。データが足りないPはその軸から除外されます。

      更新

      毎日1回、自動更新されます。

      プライバシーポリシー

      • 当サイトは、YouTube上で一般公開されている情報（動画の再生数・高評価数・チャンネルの登録者数など）のみを収集・表示しています。個人情報を収集するフォーム等はありません。
      • 動画の再生にはYouTubeの埋め込みプレーヤー（プライバシー強化モード youtube-nocookie.com）を使用しています。再生時にはYouTube（Google）のプライバシーポリシーが適用されます。
      • 当サイトは今後、アクセス解析ツールや広告配信サービス（Google AdSense等）を利用する場合があります。これらはCookieを使用して利用者の興味に応じた広告を表示することがあります。Cookieは各ブラウザの設定で無効化できます。
      • 掲載内容の修正・削除のご要望は、お手数ですが当サイトの運営者までご連絡ください。

      免責事項

      掲載データの正確性には努めていますが、YouTubeの仕様変更等により実際の数値と異なる場合があります。当サイトの情報による損害について運営者は責任を負いません。各動画・チャンネルの著作権はそれぞれの権利者に帰属します。

      About this site

      What is Vocalo Trend DB?

      An unofficial fan site that automatically collects and visualizes trends of Vocaloid (VOCALOID / SynthV / CeVIO) original songs on YouTube, updated daily.

      Who it's for

      • Vocaloid fans — check the songs growing right now, and play them right here.
      • Aspiring producers — see which vocals, lengths and title styles are trending, backed by data.

      About the data

      • Public YouTube data, collected automatically once a day.
      • Rankings list original Vocaloid songs posted within each period (week / month / year / all-time) by views. Karaoke streams, covers and medleys are excluded.
      • The Producer Directory lists channels with 10K+ subscribers or a 5M+ view Vocaloid hit, discovered daily via YouTube rankings and channel search.
      • Some view/subscriber counts are approximate; top-ranked songs use exact figures.

      What is VocaScore?

      A 0–100 score that rates the song itself, independent of channel size and post date. Instead of fixed thresholds, every song is compared to the expected value of songs in the same conditions, learned daily from all ranked songs and scored by percentile.

      • Likes (40 pts) — like ratio vs. songs with similar view counts (like ratios naturally drop as views grow, so big songs aren't penalized).
      • Reach (30 pts) — views vs. the expectation for channels of the same subscriber size. Fair for newcomers and big names alike.
      • Momentum (30 pts) — views normalized by time since posting (views ÷ days^0.65).

      Grades: S (80+) / A (65+) / B (50+) / C (35+) / D. Tap (or hover) a Score badge to see the breakdown.

      Producer Ranking axes

      About 80 axes, from straightforward (subscribers) to playful (hiragana rate in titles), computed from each producer's recent uploads, ranked songs and channel info.

      关于本站

      什么是 Vocalo Trend DB？

      每日自动收集并可视化 YouTube 上术力口（VOCALOID / SynthV / CeVIO）原创曲趋势的非官方粉丝网站。

      适合谁

      • 术力口爱好者 — 一站式查看当下热门新曲，并可直接播放。
      • 想成为P主/想火的人 — 用数据了解哪种歌声、时长、起名方式更容易火。

      关于数据

      • 每日自动收集一次 YouTube 公开信息。
      • 排行榜按播放量排序期间内（周/月/年/全期间）投稿的术力口原创曲，排除翻唱、歌回直播与串烧。
      • P主名鉴收录粉丝1万以上、或拥有500万播放神曲的频道，每日通过YouTube排行与频道搜索自动追加。
      • 部分播放量/粉丝数为约数，上榜曲使用精确数值。

      什么是 VocaScore？

      排除频道规模与投稿时间影响、为「歌曲本身」打分（0〜100）的独家评分。不用固定标准，而是与同条件歌曲的期待值比较，每日从全部上榜曲学习数据倾向，按百分位打分。

      • 好评度（40分） — 与同播放量级歌曲相比的点赞率高低。
      • 扩散力（30分） — 与同粉丝规模频道的期待值相比的播放量上浮程度。
      • 势头（30分） — 按投稿经过时间归一化的播放量（播放÷天数^0.65）。

      等级：S（80+）/ A（65+）/ B（50+）/ C（35+）/ D。点按（或鼠标悬停）Score徽章可查看明细。

      关于P主排行的维度

      从「粉丝数」到「曲名平假名率」约80种维度，由各P主的近期投稿、上榜曲与频道信息自动计算。

      사이트 소개

      Vocalo Trend DB란?

      YouTube의 보컬로이드(VOCALOID / SynthV / CeVIO) 오리지널 곡 트렌드를 매일 자동 수집·시각화하는 비공식 팬사이트입니다.

      이런 분께

      • 보컬로이드 팬 — 지금 뜨는 신곡을 모아서 체크하고 바로 재생.
      • 프로듀서 지망생 — 어떤 보컬·길이·제목이 뜨기 쉬운지 데이터로 확인.

      데이터에 대해

      • YouTube 공개 정보를 매일 1회 자동 수집합니다.
      • 랭킹은 기간 내(주/월/년/전체) 투고된 보컬로이드 오리지널 곡을 조회수순으로 정렬. 노래방송·커버·메들리는 제외.
      • 프로듀서 도감은 구독자 1만 명 이상 또는 500만 조회 히트곡 보유 채널을 수록, YouTube 랭킹·채널 검색으로 매일 자동 추가.
      • 일부 수치는 어림값이며, 랭킹 상위 곡은 정확한 값을 사용합니다.

      VocaScore란?

      구독자 수·투고일에 좌우되지 않는 「곡 자체의 평가」를 0〜100으로 내는 자체 스코어. 고정 기준이 아니라 같은 조건의 곡 기대치와 비교하여, 전체 랭크인 곡에서 매일 경향을 학습해 백분위로 점수화합니다.

      • 호감도(40점) — 같은 조회수대 곡과 비교한 좋아요율.
      • 확산력(30점) — 같은 구독자 규모 채널의 기대치 대비 조회수 상회 정도.
      • 기세(30점) — 투고 경과 시간으로 보정한 조회수(조회수÷일수^0.65).

      등급: S(80+) / A(65+) / B(50+) / C(35+) / D. Score 배지를 탭하면(PC는 마우스 오버) 내역이 보입니다.

      프로듀서 랭킹의 축

      「구독자 수」 같은 정통파부터 「곡명 히라가나율」 같은 변화구까지 약 80종. 각 프로듀서의 최근 투고·랭크인 곡·채널 정보로 자동 계산합니다.
